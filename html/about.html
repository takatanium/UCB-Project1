About this Project

The idea was to spin up a website that showcased some general information about the 50 US States in a clean and modern way.
While this project is a work in progress and far from finished, we employed a number of technologies to achieve the current build:

  • D3.js
  • Materialize.css
  • Scrollify.js
  • Google Maps API
  • Wikipedia API
  • Datausa.io API
